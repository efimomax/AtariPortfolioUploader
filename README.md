# ATARI Portfolio file uploader (USB)
Atari portfolio file uploader, using arduino and build-in filetransfer server mode.

Based on https://github.com/peterstark-code/PofoDuino_Lite


1. Connect the custom module to the Atari's parallel port.
2. Turn on the Atari.
3. On the Atari, navigate to the folder where you want to transfer the files.
4. Press the Atari + System button (see image), select File Transfer, then Receive or Server (I forgot which one :) )
5. Connect the custom module (white) to the computer via USB.
6. Launch the application on the computer (link).
7. In the application, select the serial port, the file(s), and click “Send To Portfolio.”
